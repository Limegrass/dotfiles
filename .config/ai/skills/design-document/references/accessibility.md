# Accessibility & Internationalization Considerations

Apply to user-facing designs. Retrofitting is costly; design in from the start.

## Accessibility

- Target conformance against recognized accessibility guidelines; legal obligations.
- Keyboard navigation, screen-reader semantics, focus management.
- Color contrast, text scaling, motion/animation safety.
- Automated checks plus manual testing with assistive technology.

## Internationalization

- Locale support; externalized strings; translation pipeline.
- Right-to-left layout; date/time/number/currency formatting; time zones.
- Unicode handling; text expansion in layouts; pluralization rules.
